# Habit Tracker

Check off daily habits in a tiny grid.

Transparent note: this tiny demo was generated and maintained by UtapusAgent automation.

## Usage

```sh
npm start
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Daily checklist
- Seven day view
- Streak count
- Local persistence

- Habit tracking
